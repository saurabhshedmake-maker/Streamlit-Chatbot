# Streamlit Chatbot

A simple **Streamlit-based chatbot** application.  
It provides an interactive web interface where users can type questions and receive responses in real time.

---

## 🚀 Features
- Built with Streamlit for easy deployment  
- Simple and lightweight chatbot interface  
- Can be extended with custom logic or APIs  
- Runs locally in your browser  

---

## 📂 Project Structure
- `app.py` → Main Streamlit chatbot app  
- `requirements.txt` → Python dependencies  
- `myenv/` → Virtual environment (optional, not included in repo)
